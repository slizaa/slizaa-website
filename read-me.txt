FIRST SETUP:

1. Install hugo ()

2. Install node.js ()

3. Install gulp / less
   - npm install --save-dev gulp
   - npm install --save-dev less
   - npm install --save-dev gulp-less
   - npm install --save-dev run-sequence
   - npm install --save-dev require-dir
   - npm install --save-dev del

4. Create hugo site 
   - cd ./site
   - hugo new site .



Links:
[1] http://danbahrami.io/articles/building-a-production-website-with-hugo-and-gulp-js/


After checkout:

0) npm install --global gulp-cli
1) npm install
2) Build with gulp (also runs hugo site generator):
   - <repo-dir>/gulp
3) cd ./site
5) Run the hugo server (from a second terminal):
   - hugo server --watch --verbose
